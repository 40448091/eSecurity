Unit 3
------

Generate 768-bit Diffie-Hellman parameters:
$ openssl dhparam -out dhparams.pem 768 -text

List Elliptic curves we can create:
$ openssl ecparam -list_curves

create our elliptic parameter file with:
$ openssl ecparam -name secp256k1 -out secp256k1.pem

View the details with:
$ openssl ecparam -in secp256k1.pem -text -param_enc explicit -noout

create our key pair:
$ openssl ecparam -in secp256k1.pem -genkey -noout -out mykey.pem

Unit 4
------

RSA
---

Generate a key pair with:
$openssl genrsa -out private.pem 1024	

View the RSA key pair:
$ openssl rsa -in private.pem -text 

Secure the encrypted key with 3-DES:
$openssl rsa -in private.pem -des3 -out key3des.pem

Export the public key:
$ openssl rsa -in private.pem -out public.pem -outform PEM -pubout 

Encrypt:
$ openssl rsautl -encrypt -inkey public.pem -pubin -in myfile.txt -out file.bin

decrypt with your private key:
$ openssl rsautl -decrypt -inkey private.pem -in file.bin -out decrypted.txt

----------

ECC
---

Generate a private key with:
$ openssl ecparam -name secp256k1 -genkey -out priv.pem

View the details of the ECC parameters used with:
$ openssl ecparam -in priv.pem -text -param_enc explicit -noout

Generate your public key based on your private key with:
$ openssl ec -in priv.pem -text -noout
